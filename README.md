"Ourtube"
============

Web service for watching youtube videos together online.

----------------

User guide can be found [here](https://github.com/EvgenyNasretdinov/ourtube)

Local execution
---------

If you want to use this app on your local computer,
you should run:

    git clone 

    cd ourtube

    composer install

    php artisan migrate

    php artisan serve
